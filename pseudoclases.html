pseudoclases

las pseudoclases son estilos especiales dependiendo del contexto, posicion o estado

:root

nodo principal

:hover

se presenta cuando el usuario coloca el puntero sobre alhun elemento

:active

cubre el caso en el que un elemento esta siendo activado por el usuario. Este es el orden que se debe seguir para declarar las sigs. pseudoclases si no ninguna funcionará

  1. :link {...}
  2. :visited {...}
  3. :hover {...}
  4. :active {...}

:target

representa el elemento unico, si existe alguno con un 'id' conincidente con el indentificador de fragemnto de la URL del documento.

Modal en CSS

:nth-child :firts-child :last-child

  • :nth-child: selecciona el elemento indicado que sea el enecimo hijo de su elemento padre
  • :firts-child: selecciona el elemento indicado que sea el primer hio se duelemento padre.
  • :last-child: selecciona el elemento indicado que sea el ultimo hijode su elemento padre.

Cerrar

:nth-of-type :first-of-type :last-of-type

  • :nth-of-type: selecciona el elemento indicado que sea el enecimo hijo en su tipo sin importar la jerarquia dentro del elemento
  • :first-of-type: selecciona el elemento indicado que sea el primer hijo en su tipo sin importar la jerarquia dentro del elemento padre.
  • :last-of-type: selecciona el elemento indicado que se el ultimo hijo en su tipo sin importar la jerarquia dentro del elemento padre.

:not

La pseudo-clase CSS de negación, not(x), es una notacion funcional que toma un selector simple X como argumento. Coincide con un elemento que no esta representado que no está representado por el argumento. X no debe contener otro selector de negación

:empty

corresponde a un elemento sin ningún nodo hijo. Solo cuando hay nodos de elemento o texto con uno o mas caracteres dentro del elemento ese se convierte en no vacio
